Deploy to GitHub Pages (free & permanent)
✓
GitHub Pages is free, requires no server, and gives you a permanent URL at username.github.io/repo-name
1

Create a GitHub account

Go to github.com and sign up if you haven't already
2

Create a new repository

Click "New repository" → name it portfolio-tracker → set to Public → Create
3

Upload this file

Click "uploading an existing file" → drag the index.html you downloaded → Commit changes
4

Enable GitHub Pages

Go to Settings → Pages → Source: Deploy from branch → Branch: main / (root) → Save
5

Visit your live URL

After ~1 minute, your dashboard is live at:
https://[your-username].github.io/portfolio-tracker
⚠
Note on data: Your profile data is stored in your browser's localStorage. Each device/browser will have its own data — it doesn't sync across devices automatically. To move your data, use the Export/Import buttons below.

Share with a friend

Since each browser has its own localStorage, a friend just needs to visit the same URL and create their own profile. All profiles are stored locally — completely private per device.

Stock price data

Prices are pulled from Yahoo Finance every 5 minutes while the Holdings tab is open. Canadian stocks use the .TO suffix (e.g. XEQT.TO). If prices fail to load, the app uses your last-fetched values as fallback.
